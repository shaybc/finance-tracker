# Finance Tracker (Local)

אפליקציה מקומית (NodeJS) שמנטרת תיקייה, מייבאת קבצי Excel מהבנק/כרטיסי אשראי, מנרמלת, מבצעת זיהוי כפילויות, ושומרת למסד SQLite מקומי.
ממשק WEB בעברית (RTL) עם חיפוש, פילטור, קטלוג וגרפים.

## דרישות
- Node.js 18+ (מומלץ 20+)

## התקנה
```bash
npm install
npm run db:migrate
npm run dev
```

פתח דפדפן:
- UI: http://localhost:5173
- API: http://localhost:3001

## עבודה שוטפת
העתק קבצי אקסל לתיקייה:
`./data/inbox`

המערכת:
1) תמתין שהקובץ “יתייצב” (העתקה הסתיימה)  
2) תזהה את סוג הקובץ (בנק / ויזה-פורטל / מקס)  
3) תקלוט רשומות למסד, עם מנגנון dedupe (מפתח ייחודי)  
4) תעביר את הקובץ ל־`./data/processed/<source>/<YYYY-MM>/...`

## הגדרות
העתק `.env.example` ל־`.env` לפי צורך.
